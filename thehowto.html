Essentials?

  • A github account.
  • The Github Client.
  • An IDE/code editor/text editor.

What are the steps?

  1. Make a new repository. Make sure the name of the repository is [your username].github.io. It will not work if you name it something else.
  2. Click the button set up in Desktop or open your Github client and clone the repository.
  3. Open your IDE/Code editor/text editor and make an html.index file. Write something on to it like the code below so you see a message when it works.
    4. <!DOCTYPE html>
    <html>
    <body>
    <h1>Hello world!! :DD</h1>
    </body>
    </html>
  4. Put the file into the repository folder.
  5. Commit and Publish!
  6. Check the repository on the browser and wait for it to stop pending. If your repository is showing yellow circles it means it is still pending, if it is green it's complete and your website has been update now. If it is red, you need to commit it again or commit it at another time.

And it's as easy as that! :D
